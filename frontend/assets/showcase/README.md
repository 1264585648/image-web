# ProductShot AI Showcase Assets

这些 SVG 资产内嵌了本次生成的真实商品效果图，可直接作为 `<img src="/assets/showcase/xxx.svg">` 引用。

## 推荐使用位置

| 文件 | 推荐位置 | 说明 |
| --- | --- | --- |
| `hero-before-earbuds.svg` | 首页 Hero 流程第 1 格 | 卖家原图 / Before |
| `hero-cutout-earbuds.svg` | 首页 Hero 流程第 2 格 | 抠图 / 透明背景效果 |
| `hero-white-earbuds.svg` | 首页 Hero 流程第 3 格 | 白底主图 |
| `hero-pass-earbuds.svg` | 首页 Hero 流程第 4 格 | 合规检测通过图 |
| `case-before-skincare.svg` | 案例区 Before | 普通浴室场景原图 |
| `case-after-skincare.svg` | 案例区 After | 白底商品主图 |
| `template-amazon-skincare.svg` | Amazon 白底主图模板 | 琥珀滴管瓶 |
| `template-temu-kettle.svg` | Temu 主图模板 | 绿色电热水壶 |
| `template-shopify-bag.svg` | Shopify 主图模板 | 米色帆布袋 |
| `template-transparent-bottle.svg` | 透明 PNG 模板 | 不锈钢水瓶抠图效果 |
| `template-shadow-shoe.svg` | 轻阴影棚拍模板 | 白色跑鞋 |
| `dashboard-upload-placeholder.svg` | 工作台上传占位 | 普通待处理原图 |
| `dashboard-preview-main.svg` | 工作台预览占位 | 白底主图示例 |

## 使用建议

优先替换首页 Hero 四步流程和案例 Before / After。模板缩略图可以后续在 `frontend/app.js` 的 `templateIllustrations` 中改为图片引用。
