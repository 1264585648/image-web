# ProductShot AI 设计资产

本目录用于存放 ProductShot AI（AI 商品主图工厂）的前端视觉资产与高保真设计参考。

## 当前资产

- `mockups/productshot-ai-ui-direction.svg`
  - 四个界面方向缩略图：桌面首页、桌面工作台、手机首页、手机工作台。
  - 适合作为前端开发前的视觉基准，也可以放到产品文档、PRD 或 README 中。

## 设计规范摘要

- 主色：`#0B5CFF` / `#0747D9`
- 深色文字：`#0B1736`
- 次级文字：`#60708A`
- 背景：`#F7F9FC`
- 卡片边框：`#E3E9F3`
- 成功色：`#13A05F`
- 警告色：`#F97316`
- 圆角：12px / 16px / 24px
- 阴影：轻阴影，避免重投影

## 前端落地建议

页面建议拆分为：

- Landing Page：Hero、流程卡、功能卡、模板卡、案例对比、定价、CTA
- Dashboard：上传区、预览画布、主图设置、生成结果、合规检测
- Mobile：保持单列布局，底部固定「生成主图」按钮
